# Simple Api for Music

## Configuration

Set the dotenv file

```dotenv
APP_NAME=MusicAPI
APP_ENV=dev
APP_KEY=
APP_DEBUG=true
APP_URL=http://localhost:8000

LOG_CHANNEL=stack

DB_CONNECTION=mysql
DB_HOST=maria_db
DB_PORT=3306
DB_DATABASE=musicapi
DB_USERNAME=username
DB_PASSWORD=passwd
```
